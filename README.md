### Control Code
| ALU Control Lines | Function |
| ----------------- | ---------|
| 0000              | AND      |
| 0001              | OR       |
| 0010              | add      |
| 0110              | subtract |
| 0111              | less than|
| 1100              | NOR      |
| 1111              | equals   |
